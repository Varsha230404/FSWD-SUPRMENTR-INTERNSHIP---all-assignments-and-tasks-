🖥️ Console Challenge

Master JavaScript Calculator Operations in the Browser Console

📖 How to Get Started

  1. Open this page in your web browser
  2. Press F12 or Right-Click → Inspect → Console to open Developer Tools
  3. You'll see the welcome message in the console
  4. All calculator functions are automatically loaded and ready to use!
  5. Type commands in the console to test the calculator operations
// Example: Start the console and type: calculator.add(5, 3) // Output: 8 calculator.multiply(4, 7) // Output: 28 calculator.power(2, 3) // Output: 8

📐 Available Calculator Functions

calculator.add(a, b)
Adds two numbers
>> calculator.add(10, 5)
Output: 15
calculator.subtract(a, b)
Subtracts second number from first
>> calculator.subtract(20, 8)
Output: 12
calculator.multiply(a, b)
Multiplies two numbers
>> calculator.multiply(6, 7)
Output: 42
calculator.divide(a, b)
Divides first number by second (with error handling)
>> calculator.divide(20, 4)
Output: 5
calculator.modulus(a, b)
Returns remainder of division
>> calculator.modulus(17, 5)
Output: 2
calculator.power(base, exponent)
Raises base to the power of exponent
>> calculator.power(2, 5)
Output: 32
calculator.squareRoot(num)
Calculates square root of a number
>> calculator.squareRoot(16)
Output: 4
calculator.percentage(num, percent)
Calculates percentage of a number
>> calculator.percentage(200, 15)
Output: 30
calculator.average(...numbers)
Calculates average of multiple numbers
>> calculator.average(10, 20, 30, 40)
Output: 25
calculator.factorial(n)
Calculates factorial of a number
>> calculator.factorial(5)
Output: 120
calculator.sum(...numbers)
Calculates sum of multiple numbers
>> calculator.sum(5, 10, 15, 20)
Output: 50
calculator.isEven(num)
Checks if number is even
>> calculator.isEven(4)
Output: true
calculator.isOdd(num)
Checks if number is odd
>> calculator.isOdd(7)
Output: true
calculator.isPrime(num)
Checks if number is prime
>> calculator.isPrime(17)
Output: true
calculator.temperature.celsiusToFahrenheit(celsius)
Converts Celsius to Fahrenheit
>> calculator.temperature.celsiusToFahrenheit(25)
Output: 77
calculator.temperature.fahrenheitToCelsius(fahrenheit)
Converts Fahrenheit to Celsius
>> calculator.temperature.fahrenheitToCelsius(77)
Output: 25
calculator.help()
Displays all available functions
>> calculator.help()
Output: Displays help menu
💡 Pro Tip: You can chain operations! Try calculator.add(calculator.multiply(5, 2), 10) to combine multiple calculations!
