# Weather Dashboard (React + Vite)

This assignment fetches current weather data from **WeatherAPI.com** and shows:

- weather result for a city
- loading state while request is in progress
- error state when API key is missing or request fails

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file in this project folder:

```env
VITE_WEATHER_API_KEY=your_weatherapi_key_here
```

3. Start the app:

```bash
npm run dev
```

## Build

```bash
npm run build
```
