## 🛠️ General Setup

### Prerequisites
- **Node.js** (v14 or higher) - [Download](https://nodejs.org/)
- **npm** (comes with Node.js)
- **Git** - [Download](https://git-scm.com/)
- **Code Editor** - VS Code recommended

### Installation Steps

1. **Clone the Repository:**
```bash
git clone <https://github.com/Varsha230404/FSWD-SUPRMENTR-INTERNSHIP---all-assignments-and-tasks-.git>
cd VARSHA\ S\ V\ FSWD\ SUPRMENTR\ INTERNSHIP
```

2. **For React/Vite Projects:**
```bash
cd <project-folder>
npm install
npm run dev
```

3. **For Express.js Projects:**
```bash
cd <project-folder>
npm install
node server.js
```

4. **For Plain HTML/CSS/JavaScript:**
- Simply open the `index.html` file in your web browser

## 🚀 How to Run Each Project

### Quick Start Guide by Project Type

**HTML/CSS/JavaScript Only:**
1. Locate the project folder
2. Find the `index.html` file
3. Double-click or drag to a web browser
4. Done!

**React/Vite Projects:**
1. Open terminal
2. Navigate to project: `cd project-folder`
3. Run: `npm install`
4. Run: `npm run dev`
5. Visit: `http://localhost:5173` (or provided URL)

**Express.js Projects:**
1. Open terminal
2. Navigate to project: `cd project-folder`
3. Run: `npm install`
4. Run: `node server.js`
5. Visit: `http://localhost:PORT` (check console for port number)
